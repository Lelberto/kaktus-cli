# Usage
<!-- usage -->
```sh-session
$ npm install -g kaktus-cli
$ kaktus COMMAND
running command...
$ kaktus (-v|--version|version)
kaktus-cli/1.0.0 win32-x64 node-v16.13.0
$ kaktus --help [COMMAND]
USAGE
  $ kaktus COMMAND
...
```
